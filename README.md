# IMDb Movie Rating Scraper

A Python-based scraper to fetch movie ratings, titles, and other details from IMDb. Perfect for data enthusiasts and movie lovers who want to analyze or visualize IMDb data.

## Features
- Scrape movie titles, ratings, release years, and genres.
- Save data into CSV for easy analysis.
- Easy to run and extend for personal projects or research.

## Technologies Used
- Python 3.10
- ChromeDriver
- Web-driverManager
- Requests
- Selenium
- Pandas (for data storage)
